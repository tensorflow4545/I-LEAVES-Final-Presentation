# Edu-Pad

Edu-Pad is a simple educational project that enables users to draw and write on a virtual canvas using a webcam and OpenCV. This project allows users to select different colors for drawing, including blue, black, and red, as well as an eraser function. It also features a calculator and an option to save the canvas as an image.

## Prerequisites

Make sure you have the following installed:

- Python 3.x
- OpenCV
- NumPy

## How to Use

1. Clone the repository to your local machine.
2. Open the terminal and navigate to the project directory.
3. Run the following command to execute the script:

   ```bash
   python Edu_Pad.py
Use the trackbars to adjust the color ranges and select the desired color for drawing.
Use the webcam to draw on the canvas with different colors and save the canvas when needed.
Features
Choose different colors for drawing.
Save the canvas as an image.
Erase the drawings using the eraser function.
Simple and easy-to-use interface.
